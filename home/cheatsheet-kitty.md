# Kitty Cheatsheet

## Splits
Ctrl + Shift + Enter    # New split
Ctrl + Shift + H/J/K/L  # Focus left/down/up/right

## Resize
Ctrl + Shift + Arrow    # Resize split (left/right/up/down)

## Layout
Ctrl + Shift + O        # Cycle layout

## Tabs
Ctrl + Shift + T        # New tab
Ctrl + Shift + Q        # Close tab
Ctrl + Shift + Left/Right  # Switch tab
